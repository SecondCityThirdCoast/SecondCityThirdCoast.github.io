Cook County Automated COVID-19 Vaccine Appointment Alerts


Please note - this is an unofficial alert system checking for COVID-19 vaccination appointments through various public and private providers. By being enrolled in this program, you will get alerts when appointments may be available to be booked. This program does not book appointments you must still go to the provider and book your appointment.

If the program detects possible first-dose appointment availability, you will receive a text message alerting you to check the site. Appointments are booked very quickly and may only be available for a few minutes.
It is possible that the County could alter the site in such a way that false positives may be detected or availability could be missed. I will be checking regularly to detect these changes and update the program as necessary.

This service is not meant to replace regular checking of the CCDPH page or checking with alternate providers (Walgreens, Jewel-Osco, etc.) but rather as an additional set of eyes looking for appointment availability.
